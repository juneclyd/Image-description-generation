## Краткое описание задачи

Проект посвящён задаче **Image Captioning** — автоматической генерации текстового описания изображения.

На вход системе подаётся изображение, после чего модель анализирует его содержимое и генерирует короткую текстовую подпись на английском языке. Такая система может применяться для автоматической разметки изображений, поиска по медиаконтенту, помощи людям с нарушениями зрения и анализа визуальных данных.

В проекте используется архитектура **Encoder–Decoder**:

- **Encoder** — сверточная нейронная сеть ResNet50, которая извлекает визуальные признаки из изображения;
- **Decoder** — рекуррентная сеть LSTM, которая по полученным признакам генерирует последовательность слов;
- итоговая подпись формируется последовательно: от токена `<start>` до токена `<end>` или до достижения максимальной длины описания.

## Архитектура модели

### EncoderCNN

В качестве encoder используется **ResNet50** с предобученными весами.

Особенности:

- последний классификационный слой ResNet50 удаляется;
- параметры ResNet50 замораживаются;
- изображение преобразуется в вектор признаков;
- линейный слой приводит признаки к размерности `hidden_size = 512`.

### DecoderRNN

В качестве decoder используется **LSTM**.

Параметры модели:

- `embed_size = 256`;
- `hidden_size = 512`;
- `num_layers = 1`;
- максимальная длина описания: `10` слов.

Decoder получает признаки изображения и генерирует описание по одному слову за шаг.

## Итоговые выводы

В результате работы была разработана система генерации текстовых описаний изображений.

Были выполнены следующие задачи:

- подготовлен датасет изображений и аннотаций;
- реализована модель Encoder–Decoder на основе ResNet50 и LSTM;
- проведено обучение модели;
- построены графики Train Loss и Validation Loss;
- сохранён checkpoint модели;
- разработано веб-приложение для демонстрации работы системы.

Модель способна определять основные объекты на изображении и генерировать простые текстовые описания. При этом качество описаний ограничено базовой архитектурой CNN + LSTM: иногда подписи получаются неполными или грамматически неточными.
